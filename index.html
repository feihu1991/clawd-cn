💻 安装

一行命令快速部署

🖥️ 自动安装 Node.js 和所有依赖

curl -fsSL https://clawd.org.cn/install.sh | bash
💡 中国大陆用户建议开启上方 🇨🇳 国内镜像 开关，以获得更快的下载速度。

🐋 Docker 安装

docker run -d \
  --name openclaw \
  -v ~/.openclaw:/home/user/.openclaw \
  -e ANTHROPIC_API_KEY= \
  --network host \
  openclaw/openclaw

⚙️ 手动安装

# 1. 克隆仓库
git clone https://github.com/openclaw/openclaw.git

# 2. 进入目录
cd openclaw

# 3. 安装依赖
npm install

# 4. 启动
npm start
